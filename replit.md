# YouTube Discord Notifier Bot

## Project Overview
A Discord bot that monitors YouTube channels and sends notifications to Discord channels when new videos are uploaded.

## Current Status
- Bot implementation complete
- Ready for Discord and YouTube API setup
- Supports multiple channel monitoring with subscribe/unsubscribe commands

## Setup Requirements
1. Discord bot token (from Discord Developer Portal)
2. YouTube API key (from Google Cloud Console)
3. Set both in Replit Secrets

## Commands
- `!subscribe <YOUTUBE_CHANNEL_ID>` - Subscribe to new uploads
- `!subscriptions` - View all subscriptions  
- `!unsubscribe <YOUTUBE_CHANNEL_ID>` - Unsubscribe from a channel
